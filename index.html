Un code de transfert permet à une autre personne de récupérer ce profil sur son propre compte Dokita.

Cas d'usage : enfant qui devient indépendant, mère âgée dont l'enfant prend en charge le suivi médical.

📋 Instructions :
1. Communiquez ce code à la personne concernée

3. Elle clique "Récupérer un profil" et saisit ce code
4. Le profil lui sera transféré avec tout l'historique
